TALLER #1

Una tienda de productos electrónicos desea tener un sistema para el registro de los productos que compra y vende para poder en un futuro analizar la rentabilidad de cada producto, categorías y marcas.

El cliente espera el próximo año conseguir un auspicio para abrir dos sucursales adicionales y tener una nueva bodega de almacenamiento. Indica está información pues espera que el sistema contemple esas necesidades.

El producto para su registro debe guarda los siguientes datos básicos:

Nombre, categoría, marca, código, precio de compra, bodega
Al momento de vender se debe poder registrar al menos: Local, vendedor, fecha de venta, precio de venta y reducir la cantidad de producto en bodega.

Debe permitir el registro de usuarios del sistema: Administrador, Bodeguero, Vendedor

Administrador: Crear y actualizar usuarios
Bodeguero: Crear y actualizar productos
Vendedor: Vender producto
Para guardar la información debe crear estructuras (mínimo: para usuarios, para datos de productos, para registro de ventas)
La solución debe tener al menos 3 archivos (Los nombres pueden ser cambiados):
Main.c
Funciones.h
Funciones.c
Producto: máximo 50, Usuario: máximo 10
Subir los archivos creados en un zip
Realizar un video explicando lo que hicieron en el taller (Lo realizan después)
Éxitos en sus actividades!!!
